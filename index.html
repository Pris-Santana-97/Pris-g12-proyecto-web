1. Habilidades blandas.

2. HTML Básico.

Me interesó mucho, el aprender las etiquetas y atributos, que me pueden ayudar con la estructura básica de un sítio web.

  1. instalación del editor "visual studio code"..
  2. Creación de una carppeta y un archivo "index" y la estructura básica "HTML 5"
  3. Las etiquetas de texto "h1 a h6 que son encabezados" y "p que son los párrafos".
  4. Cómo crear listas "li" listas desordenadas "ul" y ordenadas "ol" y cómo insertar una imágen "a"

3.HTML avanzado.

Fue bastante divertido. Pude darle más cuerpo a mi sítio y cada vez, se ve mejor.

  1. Ralizar contenedores con "div"
  2. Crear un encabezado "header" con una lista de navegación "nav"
  3. Hacer un pié de página "footer"
  4. Enbebér un video o un mapa "iframe"
